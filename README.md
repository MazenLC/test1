# Snakes and Ladders Game

This repository contains a simple command-line implementation of the classic Snakes and Ladders board game in Python.

## Requirements
- Python 3.7 or higher

## How to Play
Run the `snake_ladder.py` script:

```bash
python3 snake_ladder.py
```

You will be prompted to enter names for two players. Each player takes turns rolling a dice. The first player to reach square 100 wins, encountering snakes and ladders along the way.
